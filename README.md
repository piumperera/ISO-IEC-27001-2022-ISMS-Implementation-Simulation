This project simulates an Information Security Consulting engagement focused on planning and implementing an Information Security Management System (ISMS) based on the ISO/IEC 27001:2022 standard for a selected organization.

It follows a structured approach similar to real-world consulting projects, including security assessments, risk analysis, and development of key ISMS documentation.

🎯 Key Features

Conducted a Gap Analysis against ISO/IEC 27001:2022 requirements
Developed a Risk Register using asset–threat–vulnerability mapping
Created a Risk Treatment Plan aligned with Annex A controls
Prepared core ISMS documentation:
    Scope Statement
    Information Security Policy
    Statement of Applicability (SoA)
Designed a Board-Level Presentation with implementation roadmap, risks, and strategic value

🧩 Project Components
    📄 Gap Analysis Report
    📑 ISMS Documentation Pack
    📊 Executive Presentation Slides
    
👥 Team Roles
    Lead Consultant / Project Coordinator
    Risk & Controls Analyst
    ISMS Documentation Specialist
    Compliance Advisor
    Presentation & Stakeholder Engagement Lead
